# Changelog

All notable changes to the Tank Battle Game project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-22

### Added
- Complete HTML5 Canvas tank battle game
- Player tank with WASD movement controls
- Arrow key turret rotation
- Space bar shooting with muzzle flash effects
- Enemy AI tanks with pursuit behavior and shooting
- Health bar system for player and enemies
- Score tracking system
- Lives system (3 lives per game)
- Particle explosion effects
- Bullet trails and visual feedback
- Pause functionality (P key)
- Restart game button
- Modern glassmorphism UI design
- Responsive game canvas
- Keyboard controls display
- Game over and pause screens
- GitHub repository setup with documentation

### Technical
- Vanilla JavaScript game engine
- RequestAnimationFrame for smooth 60fps gameplay
- Object-oriented enemy class
- Collision detection system
- Particle system for visual effects
- Modular game loop architecture
- No external dependencies - pure HTML/CSS/JS
- Cross-browser compatible
