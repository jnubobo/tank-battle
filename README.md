# 🎮 HTML Tank Battle Game

A modern, browser-based tank battle game built with HTML5, CSS3, and vanilla JavaScript. Control your tank, defeat enemy tanks, and survive as long as possible!

![Tank Battle Game Screenshot](screenshot.png)

## ✨ Features

- **Smooth Controls**: WASD for movement, arrow keys for turret rotation, SPACE to shoot
- **Dynamic AI Enemies**: Intelligent enemy tanks that pursue and shoot at you
- **Visual Effects**: Particle explosions, bullet trails, muzzle flashes
- **Health System**: Player and enemy health bars with visual feedback
- **Score System**: Earn points for defeating enemies
- **Responsive Design**: Modern UI with glassmorphism effects
- **Pause/Restart**: Full game control with pause and restart functionality

## 🎯 How to Play

1. **Movement**: Use `W A S D` keys to move your tank
2. **Turret Control**: Use `↑ ↓ ← →` arrow keys to rotate your turret
3. **Shooting**: Press `SPACE` to fire bullets
4. **Pause**: Press `P` to pause/resume the game
5. **Restart**: Click the "Restart Game" button to start over

**Goal**: Destroy enemy tanks while avoiding their bullets. You have 3 lives - lose them all and it's game over!

## 🛠️ Technical Details

The game is built using:
- **HTML5 Canvas** for rendering
- **Vanilla JavaScript** for game logic
- **Modern CSS** with glassmorphism effects
- **RequestAnimationFrame** for smooth 60fps gameplay

### Code Structure
- `index.html` - Main HTML file with game container
- `<style>` tags - All CSS styling
- `<script>` tags - Complete game logic including:
  - Player tank controls and physics
  - Enemy AI with pursuit behavior
  - Collision detection system
  - Particle system for visual effects
  - Game state management
