# Screenshot Placeholder

This file would normally contain a screenshot of the Tank Battle Game.

To create a real screenshot:

1. Open `index.html` in your web browser
2. Play the game for a moment
3. Take a screenshot (Print Screen or browser screenshot tool)
4. Save it as `screenshot.png` in this directory

The screenshot should show:
- Player tank (green) in the bottom area
- Enemy tanks (red) scattered around
- Health bars above tanks
- Score and lives displayed
- Modern gradient background with glassmorphism UI
